Algoritmo

Un algoritmo es un conjunto de pasos ordenados y definidos que se utilizan para realizar una tarea o resolver un problema. Es fundamental en la vida cotidiana, ya que constantemente seguimos algoritmos, incluso sin darnos cuenta.
Por ejemplo, una receta de cocina es un algoritmo: seguimos una serie de instrucciones específicas y, al final, obtenemos un resultado, como un platillo preparado.


Pseudocodigo

Un pseudocódigo es una forma de escribir instrucciones para que una computadora pueda entender lo que queremos que haga, pero sin usar un lenguaje de programación complicado. Es como hacer una lista de pasos para resolver un problema, escrita de manera clara y ordenada, pero sin preocuparse por las reglas estrictas de un lenguaje de programación.
Es como explicarle a alguien cómo hacer algo, pero de manera estructurada, para que luego pueda convertirlo en un programa real.


Diagrama de Flujo

Un diagrama de flujo es una representación gráfica de un proceso o algoritmo. Se usa para mostrar, de manera visual y ordenada, los pasos que se deben seguir para realizar una tarea o resolver un problema.
